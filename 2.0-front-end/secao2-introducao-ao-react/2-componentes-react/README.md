Seção 02: Introdução ao React 

Dia 02: Componentes React 

Neste dia aprendemos:
- Criar de modo correto os componentes React. 
- Fazer uso de props corretamente. 
- Fazer composição de componentes corretamente. 
- Criar múltiplos componentes dinamicamente. 
- Utilizar PropTypes para: checar o tipo de uma prop no uso de um componente; garantir a presença de props obrigatórias no uso de um componente; checar que uma prop é um objeto de formato específico; garantir que uma prop é um array com elementos de um determinado tipo.
